L'application Corsaire

Corsaire est un GPS pour les piétons. Il permet de vous faire découvrir les lieux autour de vous et de vous y conduire. Une nouvelle façon de guidage a été développée permettant d'utiliser le vibreur du smartphone pour préciser la direction à suivre. Cette solution est particulièrement utile pour les personne ayant un handicap visuel.

Fonctionnalités

  • Recherche vocale
  • Recherche de lieux
  • Autour de moi : liste les lieux à proximités par catégorie
  • Calcul de l'itinéraire à pied
  • Calcul d'itinéraire en transport en commun (si disponible)
  • Guidage vocal
  • Multi-tâche (fonctionne lorsque l'appareil est verrouillé)
  • Guidage par vibration ( Appuyez sur la boussole puis orientez votre téléphone. Il vibre lorsque vous pointez dans la bonne direction )
  • Paramètres : désactiver la synthèse vocale, désactiver le vibreur

Fonctionnalités supplémentaires pour les aveugles

  • Position et taille des boutons adapté
  • Bouton des écouteurs pour faire répéter la synthèse vocale
  • Ou suis-je : donne l'adresse où vous vous trouvez


Toutes les fonctionnalités sont proposées gratuitement et ont pour but de vous simplifier la vie.
